![image](https://github.com/beom123456/board/assets/169109573/86f04923-0ce4-4f7c-bb89-20a6a521d76a)

## 🚀 프로젝트 개요

글을 작성하며 댓글을 달고 소통하는 게시판

## ✨ 주요 기능

- 로그인
- 글 작성
- 댓글 

## 🛠️ 기술 스택

- **프론트엔드:** HTML, CSS, JavaScript
- **백엔드:** Spring legacy, JAVA 
- **데이터베이스:** mySQL
- **버전 관리:** Git & GitHub

### 사전 요구 사항

- Java 11 이상
- sqldeveloper
